# 非サポート機能一覧

Tsuchinokoトランスパイラが現在サポートしていないPython機能の一覧です。

## 言語機能

- **名前付き引数** (`func(name="value")`)
- **可変長引数** (`*args`, `**kwargs`)
- **デフォルト引数値** (`def func(x=10)`)
- **デコレータ** (`@staticmethod`以外)
- **ジェネレータ** (`yield`文)
- **Async/Await** 構文
- **コンテキストマネージャ** (`with`文)
- **多重継承**
- **メタクラス**
- **セイウチ演算子** (`:=`)

## 例外処理

- **try/except/finally** ブロック
- **カスタム例外クラス**
- **例外チェーン** (`raise ... from ...`)

## クラス機能

- **クラス継承** (基本構造体的クラス以外)
- **プロパティ** (`@property`)
- **クラスメソッド** (`@classmethod`)
- **`__repr__`, `__str__`** その他マジックメソッド
- **演算子オーバーロード** (`__add__`, `__eq__`など)
- **抽象基底クラス**

## 組み込み型

- **Set型** (`set[T]`)
- **Frozenset**
- **複素数**
- **Bytes/Bytearray**
- **Decimal**

## 標準ライブラリ

- ほとんどの標準ライブラリモジュールは非サポート
- **ファイルI/O** (`open()`、ファイル操作)
- **正規表現** (`re`モジュール)
- **JSONパース** (`json`モジュール)
- **日時** (`datetime`モジュール)
- **コレクション** (`collections`モジュール、dict以外)

## 高度な機能

- **複数代入** (`a, b, c = 1, 2, 3` - インデックススワップのみ部分サポート)
- **スター式** (`first, *rest = [1, 2, 3]`)
- **辞書内包表記** (`{k: v for k, v in ...}`)
- **セット内包表記** (`{x for x in ...}`)
- **Global/Nonlocal** 文
- **Import文**
- **モジュールシステム**

## 注記

ここに記載された機能は将来のバージョンで追加される可能性があります。機能リクエストはGitHubリポジトリのissueにお願いします。
