# サポート機能一覧

Tsuchinokoトランスパイラが現在サポートしているPython機能の一覧です。

## 基本構文

- **変数宣言** 型ヒント付き (`x: int = 10`)
- **基本型**: `int`, `float`, `str`, `bool`, `None`
- **コレクション型**: `list[T]`, `dict[K, V]`, `tuple[...]`
- **Optional型**: `Optional[T]`, `T | None`
- **算術演算子**: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- **比較演算子**: `==`, `!=`, `<`, `>`, `<=`, `>=`
- **論理演算子**: `and`, `or`, `not`
- **累算代入演算子**: `+=`, `-=`, `*=`, `/=`, `//=`, `%=`

## 制御フロー

- **If/elif/else** 文
- **Forループ** (`range()`、コレクション反復)
- **Whileループ**
- **Break/Continue** 文
- **条件式** (`x if cond else y`)

## 関数

- **関数定義** 型ヒント付き
- **Return文** オプショナル値付き
- **再帰** サポート
- **ネスト関数** (Rustクロージャに変換)
- **Lambda式** (`lambda x: x + 1`)
- **高階関数** (関数を引数として渡す)

## データ構造

- **リストリテラル** と操作
- **リスト内包表記** (基本と条件付き)
- **辞書リテラル** と操作
- **タプルリテラル** とアンパック
- **構造体定義** (クラス構文経由)
- **負のインデックス** (`nums[-1]`)
- **スライス記法** (`[:3]`, `[-3:]`, `[1:n-1]`)
- **インデックススワップ** (`a[i], a[j] = a[j], a[i]` → `a.swap()`)

## クラス & オブジェクト

- **基本クラス定義** `__init__`付き
- **インスタンス属性** (`self.attr`)
- **メソッド定義**
- **静的メソッド** (`@staticmethod`)

## 組み込み関数

- `len()` - 長さ取得
- `range()` - 数値範囲反復
- `print()` - コンソール出力
- `list()` - リスト変換
- `min()`, `max()` - 最小/最大値
- `abs()` - 絶対値
- `int()`, `float()`, `str()`, `bool()` - 型変換

## 文字列機能

- **文字列リテラル** (シングル/ダブルクォート)
- **F文字列** (`f"Hello {name}"`)
- **文字列メソッド**: `.upper()`, `.lower()`, `.strip()`, `.split()`, `.join()` など

## エラー処理

- **raise** 文 (`panic!`に変換)
- **ValueError**, **TypeError** (`panic!`に変換)

## 型システム

- **型エイリアス** (`MyType = list[int]`)
- **Callable型** (`Callable[[T], U]`)
- **関数型推論**
- **自動型強制** (Auto-Ref, Auto-Deref, Auto-Clone)
